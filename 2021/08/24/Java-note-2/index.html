ж ҮиҜҶз¬ҰдёҺе…ій”®еӯ—

е…ій”®еӯ—

Java иҜӯжі•дёӯиҮӘеёҰзҡ„еҚ•иҜҚпјҢеҰӮ publicпјҢintпјҢpackageзӯү

ж ҮиҜҶз¬Ұ

зұ»еҗҚгҖҒеҸҳйҮҸеҗҚд»ҘеҸҠж–№жі•еҗҚйғҪиў«з§°дёәж ҮиҜҶз¬ҰпјҢж ҮиҜҶз¬ҰдёҚиғҪд»Ҙе…ій”®еӯ—е‘ҪеҗҚ

  • ж ҮиҜҶз¬Ұд»ҘеӨ§е°ҸеҶҷеӯ—жҜҚгҖҒзҫҺе…ғз¬ҰеҸ·пјҲ$пјүгҖҒдёӢеҲ’зәҝпјҲ_пјүејҖеӨҙ
  • йҰ–еӯ—з¬ҰеҗҺеҸҜд»ҘдҪҝз”ЁеӨ§е°ҸеҶҷеӯ—жҜҚгҖҒзҫҺе…ғз¬ҰеҸ·гҖҒдёӢеҲ’зәҝгҖҒж•°еӯ—
  • ж ҮиҜҶз¬ҰеӨ§е°ҸеҶҷж•Ҹж„ҹ
  • дёҚе»әи®®дҪҝз”Ёдёӯж–ҮдёҺжӢјйҹіе‘ҪеҗҚ

ж•°жҚ®зұ»еһӢ

еҹәжң¬зұ»еһӢ

  • ж•°еҖјзұ»еһӢ

    • ж•ҙж•°зұ»еһӢ

      • byteпјҡеҚ 1дёӘеӯ—иҠӮ
      • shortпјҡеҚ 2дёӘеӯ—иҠӮ
      • intпјҡеҚ 4дёӘеӯ—иҠӮ
      • longпјҡеҚ 8дёӘеӯ—иҠӮ

      дәҢиҝӣеҲ¶ 0bпјҢе…«иҝӣеҲ¶0пјҢеҚҒе…ӯиҝӣеҲ¶0x

    • жө®зӮ№зұ»еһӢ

      • floatпјҡеҚ 4дёӘеӯ—иҠӮ
      • doubleпјҡеҚ 8дёӘеӯ—иҠӮ

      жө®зӮ№ж•°жңүиҜҜе·®пјҢжңҖеҘҪдёҚиҰҒдҪҝз”Ёжө®зӮ№ж•°иҝӣиЎҢжҜ”иҫғж“ҚдҪң

    • еӯ—з¬Ұзұ»еһӢ

      • charпјҡеҚ 2дёӘеӯ—иҠӮ

      unicodeзј–з Ғ еҚ 2еӯ—иҠӮ

      иҪ¬д№үеӯ—з¬Ұпјҡ\t \n зӯү

  • booleanзұ»еһӢпјҡеҚ дёҖдҪҚпјҢеҸӘжңү trueе’ҢfalseдёӨз§ҚеҸ–еҖј

еј•з”Ёзұ»еһӢ

  • зұ»
  • жҺҘеҸЈ
  • ж•°з»„

зұ»еһӢиҪ¬жҚў

byte, short, char -> int -> long -> float -> double

дёҖиҲ¬д»Һе°ҸеҲ°еӨ§иҪ¬жҚў

ејәеҲ¶зұ»еһӢиҪ¬жҚў

egпјҡ

int i = 10;
byte b = (byte)i;

й«ҳиҪ¬дҪҺз”ЁејәеҲ¶иҪ¬жҚўпјҢжіЁж„ҸеҶ…еӯҳжәўеҮәдёҺзІҫеәҰй—®йўҳ

иҮӘеҠЁзұ»еһӢиҪ¬жҚў

egпјҡ

char c = 'a';
int i = c+1;

дҪҺеҲ°й«ҳз”ЁиҮӘеҠЁиҪ¬жҚў

еңЁи®Ўз®—жҜ”иҫғеӨ§зҡ„ж•°еӯ—ж—¶жіЁж„ҸжәўеҮәзҡ„й—®йўҳ

еҸҳйҮҸгҖҒеёёйҮҸгҖҒдҪңз”Ёеҹҹ

еҸҳйҮҸ

жңҖеҹәжң¬зҡ„еӯҳеӮЁеҚ•е…ғпјҢеҢ…жӢ¬еҸҳйҮҸеҗҚгҖҒеҸҳйҮҸзұ»еһӢе’ҢдҪңз”Ёеҹҹ

  • жҜҸдёӘеҸҳйҮҸйғҪжңүзұ»еһӢпјҢеҸҜд»ҘжҳҜеҹәжң¬зұ»еһӢпјҢд№ҹеҸҜд»ҘжҳҜеј•з”Ёзұ»еһӢ
  • еҸҳйҮҸеҗҚдёәеҗҲжі•зҡ„ж ҮиҜҶз¬Ұ

еёёйҮҸ

еҲқе§ӢеҢ–еҗҺдёҚеҶҚж”№еҸҳпјҢдёҖиҲ¬з”Ёfinal+зұ»еһӢ+еёёйҮҸеҗҚе®ҡд№ү

final double PI = 3.14

еёёйҮҸеҗҚдёҖиҲ¬з”ЁеӨ§еҶҷеӯ—з¬Ұ

дҪңз”Ёеҹҹ

  • зұ»еҸҳйҮҸпјҡдҪңз”ЁеҹҹжӣҙеӨ§
  • е®һдҫӢеҸҳйҮҸпјҡд»ҺеұһдәҺеҜ№иұЎпјҢжңүй»ҳи®ӨеҖј
  • еұҖйғЁеҸҳйҮҸпјҡдҪңз”ЁдәҺж–№жі•еҶ…пјҢйңҖиҰҒеҲқе§ӢеҢ–

еҸҳйҮҸе‘ҪеҗҚи§„иҢғ

  • и§ҒеҗҚзҹҘж„Ҹ
  • зұ»еҸҳйҮҸпјҡйҰ–еӯ—жҜҚе°ҸеҶҷе’Ңй©јеі°еҺҹеҲҷпјҡmonthSalary
  • еұҖйғЁеҸҳйҮҸпјҡйҰ–еӯ—жҜҚе°ҸеҶҷе’Ңй©јеі°еҺҹеҲҷ
  • еёёйҮҸпјҡеӨ§еҶҷеӯ—жҜҚе’ҢдёӢеҲ’зәҝ
  • зұ»еҗҚпјҡйҰ–еӯ—жҜҚеӨ§еҶҷе’Ңй©јеі°еҺҹеҲҷпјҡHelloWorld
  • ж–№жі•еҗҚпјҡйҰ–еӯ—жҜҚе°ҸеҶҷе’Ңй©јеі°еҺҹеҲҷпјҡsetName

иҝҗз®—з¬Ұ

  • з®—жңҜиҝҗз®—з¬Ұпјҡ+ , - , * , / , % , ++ , вҖ“

  • иөӢеҖјиҝҗз®—з¬Ұпјҡ=

  • е…ізі»иҝҗз®—з¬Ұпјҡ> , < , >= , <= , == , != , instaceof

    instanceofжҳҜJavaдёӯзҡ„дәҢе…ғиҝҗз®—з¬ҰпјҢе·Ұиҫ№жҳҜеҜ№иұЎпјҢеҸіиҫ№жҳҜзұ»пјӣеҪ“еҜ№иұЎжҳҜеҸіиҫ№зұ»жҲ–еӯҗзұ»жүҖеҲӣе»әеҜ№иұЎж—¶пјҢиҝ”еӣһtrueпјӣеҗҰеҲҷпјҢиҝ”еӣһfalse

  • йҖ»иҫ‘иҝҗз®—з¬Ұпјҡ&&пјҢ||пјҢ !

  • дҪҚиҝҗз®—з¬Ұпјҡ&пјҢ|пјҢ^пјҢ~пјҢ>>пјҢ<<пјҢ>>>

  • жқЎд»¶иҝҗз®—з¬Ұпјҡ? :

  • жү©еұ•иөӢеҖјиҝҗз®—з¬Ұпјҡ+=пјҢ-=пјҢ*=пјҢ/=
